Injectable(descriptor) → {function}

A decorator which registers the class it is attached to.

Parameters:
Name Type Description
descriptor object

configuration needed for the entity being registered

Properties
Name Type Description
name string

entity name, used to access or inject the entity
ns string

namespace or prefix of the reference name separated with a '.' (e.g: widget.foo)
params array | object

list of parameters used to initialse/call the entity if none are given
inject array

list of entity names to be injected
inherit string

descriptor properties of the referenced entity are used as defaults
singleton boolean

turn the entity into a singleton
role string

role of the entity (e.g: Service or Component)
accept array

list of roles which are allowed to be injected
reject string

list or roles which are not allowed to be injected
action number

the action to be applied to the entity when requested
